# 🚦 Traffic Light Controller

## 📌 Project Description
This is a **Digital Logic Design** project. A **Traffic Light Controller** is designed using **Arduino UNO** on **Tinkercad**. The system controls three LEDs (Red, Yellow, Green) in a timed sequence to simulate real-world traffic signals.

## 🛠️ Components Used
- Arduino UNO R3
- LEDs (Red, Yellow, Green)
- Resistors (220Ω)
- Breadboard
- Jumper Wires
- Tinkercad Simulator

## ⏱️ Timing Sequence
| Light | Duration |
|-------|----------|
| 🟢 Green | 5 seconds |
| 🟡 Yellow | 2 seconds |
| 🔴 Red | 5 seconds |
